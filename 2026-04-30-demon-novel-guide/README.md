# 别叫我恶魔资料展示页

这是一个静态网页原型，用来展示《别叫我恶魔》的剧情篇章、世界观设定、阵营人物和关键地点。页面不会把长篇整理稿整段堆进去，而是以左侧资料库导航、可展开卡片、筛选表格和人物详情弹窗呈现。

## 打开方式

直接在浏览器打开：

`/Users/dysania/program/best 2-hour build/2026-04-30-demon-novel-guide/index.html`

## 当前内容

- 剧情篇章：18 个主线篇章，包含章节范围、核心人物和展开详情。
- 世界观设定：基础世界、力量体系、种族、组织、地点、计划与终局结构。
- 人物阵营：按主角团、大夏、学院同代、魔域、神族、妖族、灵族、星空终局等分类。
- 视觉素材：使用 imagegen 生成的网页设计稿与世界观概念图，并保留页面内生成的风格化人物头像。
